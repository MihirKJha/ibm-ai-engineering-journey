# 📖 Overview

This capstone project consolidates the knowledge gained throughout the IBM AI Engineering Professional Certificate by solving a real-world **binary image classification** problem.

The objective is to classify satellite imagery into:

- 🌾 Agricultural Land
- 🏙️ Non-Agricultural Land

Using both **Keras** and **PyTorch**, the project explores multiple deep learning approaches, beginning with scalable data pipelines, progressing through Convolutional Neural Networks (CNNs), and culminating with **Vision Transformers (ViTs)** and **CNN–ViT integration** for geospatial image classification.

Unlike previous modules that focused on individual concepts, this capstone demonstrates the complete Deep Learning engineering lifecycle—from data preparation to model evaluation and architectural comparison.

# 🌍 Project Overview

**Business Problem**

A fertilizer company wants to identify potential agricultural land in new regions using satellite imagery.

Instead of manually reviewing thousands of images, Deep Learning models automatically classify land into agricultural and non-agricultural categories.

This demonstrates a practical AI solution applicable to:

- Precision Agriculture
- Climate Monitoring
- Urban Planning
- Land Cover Analysis
- Remote Sensing
