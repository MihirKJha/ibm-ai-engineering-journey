# 📖 Overview

This directory contains the hands-on labs, practice project, and final project completed during the **Deep Learning with Keras and TensorFlow** course from the **IBM AI Engineering Professional Certificate** on **Coursera**.

The course focuses on building practical Deep Learning applications using TensorFlow and Keras. It covers modern neural network architectures, custom model development, transfer learning, computer vision, transformers, generative models, reinforcement learning, hyperparameter tuning, and production-ready Deep Learning workflows.
