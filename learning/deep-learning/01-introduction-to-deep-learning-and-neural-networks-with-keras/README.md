# 📖 Overview

This directory contains the hands-on labs, Jupyter notebooks, and final project completed during the **Introduction to Deep Learning & Neural Networks with Keras** course from the **IBM AI Engineering Professional Certificate** on **Coursera**.

The course introduces the fundamental concepts of Deep Learning, including Artificial Neural Networks, Forward and Backpropagation, Gradient Descent, Activation Functions, and modern neural network architectures such as CNNs, RNNs, Transformers, and Autoencoders. It also demonstrates how to build regression and classification models using Keras.
