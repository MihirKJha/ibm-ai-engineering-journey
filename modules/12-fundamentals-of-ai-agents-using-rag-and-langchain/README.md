# Fundamentals of AI Agents using RAG and LangChain

> A foundational guide to **AI Agents**, **Retrieval-Augmented Generation (RAG)**, **Prompt Engineering**, **In-Context Learning**, and **LangChain**, introducing the core concepts required to build intelligent AI-powered applications.

This module is based on the **IBM AI Engineering Professional Certificate** and serves as an introduction to modern AI application development. It bridges the gap between **LLM Engineering** and **building AI-powered applications** using RAG and LangChain.

---

# Module Overview

Large Language Models (LLMs) are powerful language understanding and generation models, but they have limitations such as outdated knowledge, hallucinations, and the inability to interact with external tools.

This module introduces the foundational concepts that enable LLMs to become practical AI applications through:

- AI Agents
- Retrieval-Augmented Generation (RAG)
- Prompt Engineering
- In-Context Learning
- LangChain

The focus is on understanding the **fundamental concepts and workflows** rather than advanced implementation techniques.

---

# Learning Objectives

After completing this module, you will be able to:

- Explain the fundamentals of AI Agents.
- Understand the motivation behind Retrieval-Augmented Generation (RAG).
- Apply basic Prompt Engineering and In-Context Learning techniques.
- Understand the core components of the LangChain framework.
- Describe how modern AI applications combine LLMs, retrieval, and external tools.
