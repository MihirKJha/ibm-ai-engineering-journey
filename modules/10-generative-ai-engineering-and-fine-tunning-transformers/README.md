# Generative AI Engineering and Fine-Tuning Transformers

This repository contains the hands-on labs, notes, and supporting materials completed as part of the **Generative AI Engineering and Fine-Tuning Transformers** course from the **IBM AI Engineering Professional Certificate**.

The course focuses on adapting pretrained Transformer models for domain-specific tasks using modern fine-tuning techniques. It covers the Hugging Face ecosystem, PyTorch-based training, parameter-efficient fine-tuning (PEFT), LoRA, QLoRA, and model optimization techniques used in production AI systems.
