# 📖 Overview

This directory contains the hands-on labs, Jupyter notebooks, module assessment projects, and final project completed during the **Machine Learning with Python** course from the **IBM AI Engineering Professional Certificate** on **Coursera**.

The course provides a practical introduction to the complete Machine Learning lifecycle—from understanding core algorithms and preparing data to building, evaluating, validating, and optimizing Machine Learning models using **Python** and **Scikit-learn**.
